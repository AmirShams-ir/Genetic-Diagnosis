# Code of Conduct

## Our Pledge

We are committed to providing a welcoming and respectful environment for
everyone.

## Expected Behavior

- Be respectful.
- Be constructive.
- Welcome newcomers.
- Focus on improving the project.

## Unacceptable Behavior

- Harassment
- Personal attacks
- Offensive language
- Discrimination

## Enforcement

Project maintainers may remove comments, issues or pull requests that
violate this Code of Conduct.
