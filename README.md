# Genetic Diagnosis System

![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-blue)
![Language](https://img.shields.io/badge/VB.NET-.NET_Framework-purple)

A professional clinical software platform for **Genetic Disease Diagnosis**
designed to assist medical geneticists in recording clinical findings,
managing patient information, and supporting differential diagnosis of
genetic disorders.

Originally developed as an academic research project at
**Shahrekord University**, Iran.

---

# Features

- Clinical Patient Records
- Medical History Management
- Clinical Signs & Symptoms
- Laboratory Test Recording
- Differential Diagnosis Support
- Genetic Disease Database Integration
- OMIM Integration
- Medical Image Management
- User Authentication
- Persian User Interface
- Local Database
- Modular Architecture

---
