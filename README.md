# Genetic Counseling System

![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)

A professional software platform for **Genetic Counseling**, patient
management, pedigree analysis and genetic consultation.

Originally developed as part of an academic research project at
**Shahrekord University**.

---

## Features

- Patient Management
- Genetic Counseling
- Pedigree Recording
- Medical Reports
- Image Management
- Local Database
- User Authentication
- Persian User Interface

---

## Project Status

Legacy VB.NET Version

Future versions will be developed using modern technologies.
