# سامانه تشخیص بیماری‌ها و سندرم‌های ژنتیکی

![License](https://img.shields.io/badge/License-Apache%202.0-blue.svg)
![Platform](https://img.shields.io/badge/Platform-Windows-blue)
![Language](https://img.shields.io/badge/VB.NET-.NET_Framework-purple)

سامانه تشخیص بیماری‌های ژنتیکی یک نرم‌افزار تخصصی جهت ثبت اطلاعات بالینی بیماران، مدیریت پرونده‌های پزشکی و کمک به تشخیص افتراقی بیماری‌ها و سندرم‌های ژنتیکی می‌باشد.

این نرم‌افزار در ابتدا به عنوان بخشی از یک پروژه پژوهشی دانشگاهی در **دانشگاه شهرکرد** طراحی و توسعه یافته است.

---

# امکانات

- ثبت پرونده بیماران
- ثبت شرح حال پزشکی
- ثبت علائم و یافته‌های بالینی
- ثبت نتایج آزمایش‌های ژنتیکی
- کمک به تشخیص افتراقی بیماری‌ها
- ارتباط با پایگاه داده OMIM
- مدیریت تصاویر پزشکی
- سیستم ورود کاربران
- رابط کاربری فارسی
- پایگاه داده محلی
- معماری ماژولار

---

# فناوری‌های استفاده شده

- Visual Basic .NET
- .NET Framework
- Microsoft Access Database
- Windows Forms

---

# وضعیت پروژه

نسخه موجود، نسخه کلاسیک مبتنی بر VB.NET می‌باشد.

این مخزن نسخه اصلی نرم‌افزار را حفظ می‌کند.

نسخه‌های آینده با فناوری‌های مدرن توسعه خواهند یافت، از جمله:

- Python
- FastAPI
- PostgreSQL
- Vue.js
- هوش مصنوعی
- رابط کاربری مدرن چندسکویی
